Dota 2 Party Maker Bot
A Discord bot designed for creating "LFG" (Looking For Group) announcements for Dota 2.

Features:
Mode Selection: Choose between Ranked, Normal, and other game modes.

Automatic Thread Creation: Generates dedicated chat threads for every group.

Auto-Cleanup: Announcements are automatically deleted after 20 minutes to keep channels tidy.

Thread Management: Automatically removes old threads after 5 days.
